# Timer Pijat Sederhana

Halaman web sederhana untuk menghitung waktu pijat.

Fitur:
- Set durasi (menit)
- Tombol cepat 5/10/15/30 menit
- Start / Pause / Reset
- Bunyi dan getar ketika waktu habis

Cara pakai:
1. Buka `index.html` di browser (cukup klik dua kali atau pakai simple HTTP server).
2. Atur durasi, tekan Start.
3. Ketika selesai akan ada bunyi dan notifikasi (alert).
